## About Technical Creators Club

**Technical Creators Club** (*Lithuanian: Tech Kūrėjų Klubas*) is a community blog about the tools, techniques, and ideas that make building software better.

Posts are practical and direct — no padding, no SEO fluff. Just the useful part.

### Topics

- **Git & Version Control** — workflows, tips, power features
- **Developer Tools** — CLIs, editors, productivity utilities
- **Architecture** — patterns, design decisions, tradeoffs
- **AI & Automation** — what's actually useful in practice
- **Open Source** — contributing, maintaining, community

### How the Blog Works

Every post is a Markdown file in `_posts/`. The entire blog lives in a Git repository — versioned, forkable, no CMS required.

Want to contribute a post? See the [contribution guide](https://github.com/Technical-Creators-Club/Technical-Creators-Club.github.io/blob/main/CONTRIBUTING.md).
